Display and Visibility

It does not completely work in Netscape 4, Opera and Omniweb because those browsers don't support the changing of display.

The changing of display works in iCab, but not the changing of visibility.

Opera 5 made the invisibility style ('visible' or 'hidden') upper case, so that reading it out may cause confusion. Bug solved in Opera 6 (5b5 on Mac). The script does no longer take it into account.

This is a test page to clarify the difference between display and invisibility. Below you can experiment with invisibility and blocking.

For an example of a script using only invisibility, see the Navigation: layers script, for an example of a script using only display, see the Navigation: display script.
